# Тестирование API микросервиса объявлений

## Описание проекта
Автоматизированные тесты для проверки REST API микросервиса объявлений Avito.

## Функциональность тестируемого сервиса
- Создание объявлений
- Получение объявления по ID
- Получение всех объявлений по sellerId
- Получение статистики по itemId

## Предварительные требования
- Python 3.8+
- pip (пакетный менеджер Python)

## Установка и настройка

1. **Клонируйте репозиторий** (если применимо) или сохраните файлы в нужную директорию

2. **Установите зависимости:**
